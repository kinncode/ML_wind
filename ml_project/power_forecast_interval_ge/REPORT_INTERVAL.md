## 2. 區間發電能量預測核心優勢 (100% 排除當前時步洩漏)
將預測標的調整為未來區間 $[t+1 .. t+k]$ 平均功率並嚴格對齊無洩漏遮罩後：
1. **消弭高頻陣風與湍流雜訊**：3 小時區間平均功率預測擬合度 $R^2 = 0.914$。
2. **對齊國際標準**：採用容量正規化 nRMSE (nRMSE_cap = RMSE / 1.0)。
3. **對稱式 Persistence 基準**：以過去 H 小時滾動均值預測未來 H 小時區間均值，比單純瞬時值更嚴格。
